## 如何开始

+ clone项目到本地

```git
$ git clone https://github.com/immisso/blog-web
```

+ 安装包

```bash
$ npm install
```
或者
```bash
$ yarn
```
+ 启动项目

```bash
$ npm run start
```
或
```bash
$ yarn start
```
启动成功后，然后再浏览器上打开[http://localhost:8888](http://localhost:8888)即可！
